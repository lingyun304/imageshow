# PromptMedia - 媒体分类展示网站需求文档 (PRD)

## 1. 项目背景与目标
随着 ComfyUI 和 Stable Diffusion 等 AI 创作工具的普及，设计师与多媒体创作者累积了海量的媒体文件（包含图片、音频和视频）。这些图片的 PNG/WebP 元数据中嵌入了极具价值的“生成参数”（工作流、提示词等），且音视频多媒体的管理也需要更优雅的呈现方式。
本项目旨在构建一个**完全去后端的、视觉效果极其 premium 的静态媒体展示系统**，让用户通过简单的本地文件夹分类组织媒体文件，并自动解析和展示图片中的 ComfyUI 生成细节，以及播放视频和音频，方便内容管理、技术重现 and 成果分享。

---

## 2. 系统技术架构
为了实现无后端的全静态运行并支持本地文件同步，系统采用以下架构：
- **静态资源扫描器 (Build-time CLI):** 使用 Node.js 编写的 `scan-media.js` 脚本。运行时扫描本地 `/public/media/`（该文件夹可作为指向本地其他任意媒体输出目录的软链接/快捷方式），解析 PNG (`tEXt`/`iTXt`) 或 WebP (`XMP`) 二进制数据块中的 ComfyUI 提示词数据，并提取侧边栏 `.json` 文件作为兼容格式。生成最终的 `/public/media/media-data.json` 索引文件。
- **现代化前端展现层 (React Single Page App):** 采用 React 19 + Vite 8 + Vanilla CSS 构建，纯静态展示，通过 `fetch()` 读取扫描生成的 JSON 数据库。
- **UI/UX 设计语言:** 支持三套独立的视觉系统一键无缝切换：默认的【科技暗黑 (Cyber Dark)】、亮色温暖且具有春意感的【清新雅致 (Fresh Mint)】、以及搭载了动感渐变背景与丰富微交互的【炫酷动感 (Dynamic Anime)】。三套主题均融入了高品质玻璃态拟物化 (Glassmorphic) 模块、响应式瀑布流 (Masonry Grid) 和顺滑的过渡动画，提供极佳的感官体验。

---

## 3. 功能需求规格

### 3.1 首页介绍模块 (Home)
- **品牌 Hero 区:** 采用高辨识度的标题、大字号渐变文本和科技微动效，确立高级、现代的整体基调。
- **实时统计看板:** 动态显示已解析的本地图片总数、分类统计数、以及 ComfyUI 图像元数据解析覆盖率。
- **核心卖点卡片:** 突出“零后端托管”、“自动文件夹分类”、“ComfyUI 流程提取”等主要特性。
- **分类导航预览:** 自动渲染每个分类的卡片（展示中英文双语名，如 赛博朋克 (Cyberpunk)），背景显示该分类下的首张图片，点击可瞬间跳转到对应画廊。

### 3.2 分类媒体库模块 (Gallery)
- **多维度检索与搜索:**
  - 支持对正向提示词、负向提示词、模型名、采样器和文件名的模糊搜索。
  - 支持按生成所用大模型（Checkpoint）、采样器（Sampler）等参数的多选组合筛选。
  - 支持仅过滤显示“有 ComfyUI 生成数据”的图片。
- **动态分类 Tab 栏:** 根据文件夹层级自动生成分类选项卡，支持中英文双语对照显示（如 赛博朋克 (Cyberpunk)），并带有一键清除所有筛选功能。
- **瀑布流展示 (Masonry Layout):** 图片以等宽、不等高的响应式瀑布流卡片平铺，卡片上的分类徽章亦采用中英文双语展示，悬浮时露出快速参数卡与“查看生成参数”按钮。

### 3.3 参数详情查看器 (Detail Modal)
- **超限遮罩与按键导航:** 模糊背景弹出层，支持使用键盘左右方向键 (←/→) 切换上一张/下一张图片，支持 `Esc` 退出。
- **极清预览与下载:** 左侧展示大图，支持自适应缩放，提供原图一键下载。
- **正/负向提示词视窗:** 卡片式展示提示词，包含一键复制按钮与绿色成功状态提示。
- **参数矩阵宫格:** 规范整理展示 Steps、CFG、Seed、Sampler、Scheduler、Denoise、Checkpoint 以及分辨率与文件属性。
- **源码级工作流查看器 (Tabs):** 
  - **Prompt API Graph:** 提取并美化展示底层的 API 格式节点网络。
  - **Workflow UI Graph:** 提取展示 ComfyUI 界面布局连线数据。
  - 均支持一键复制，用户可直接贴回 ComfyUI 恢复生成场景。

### 3.4 部署说明页 (Guide)
- 集成在网站内部的部署使用文档，包含本地扫描流程、文件夹结构图、元数据提取原理解释和各平台静态部署指引，自带锚点侧边导航。

### 3.5 多主题视觉系统 (Theme System)
为了契合不同用户的审美并适配不同的生成图片风格，系统设计并实现了三套独立的视觉风格，支持在导航栏右侧一键切换：
1. **科技暗黑 (Cyber Dark - 默认):**
   - **设计基调:** 赛博朋克与科幻感。
   - **色彩系统:** 极黑背景 (`#07090e`)、深蓝紫色卡片、紫色 (`#8b5cf6`) 和青色 (`#06b6d4`) 霓虹流光。
   - **核心视觉:** 高反差暗度、玻璃磨砂、明亮的边框发光，体现出高级且冷静的工业范。
2. **清新雅致 (Fresh Mint):**
   - **设计基调:** 纯净、自然、明亮与简约。
   - **色彩系统:** 温暖的浅绿乳白色背景 (`#f5faf7`)、纯白卡片背景、森林薄荷绿 (`#10b981`) 和天蓝色 (`#0ea5e9`) 辅助色。
   - **核心视觉:** 柔和边缘投影、淡绿细微边框、春意盎然的护眼色调，适合日式或大自然类生成图展示。
3. **炫酷动感 (Dynamic Anime):**
   - **设计基调:** 动漫次元、幻彩与强烈的感官动效。
   - **色彩系统:** 深紫色基底、流光溢彩的霓虹艳粉 (`#ec4899`) 与赛博金黄 (`#eab308`)。
   - **核心视觉:** 
     - **动态渐变背景:** 四色流光渐变（`linear-gradient`）缓慢平滑循环滚动，具有呼吸感。
     - **浮动粒子动效:** 首页 Hero 区域渲染多组随机缓动上浮的毛玻璃幻影气泡。
     - **卡片倾斜动画:** 卡片悬浮时触发 3D 旋转偏转（`rotate(0.6deg)`）与弹性微放大，极具动感。
     - **按钮抖动与闪烁:** 核心操作按钮及分类 Tab 享有炫目的呼吸阴影和微抖动（`shake`）反馈。

### 3.6 提示词生成器模块 (Prompt Generator)
为了方便创作者构建高质量的提示词，系统集成了一个强大的 Danbooru 提示词生成器：
- **画质前缀预设 (Model Presets):**
  - **Pony Diffusion V6:** 自动带入通用的 `score_9, score_8_up, score_7_up...` 高画质评分前缀与 `source_anime` 标签，以及契合的分级标签。
  - **Illustrious XL:** 支持角色与版权（Character/Copyright）标签强制置前，分级标签紧跟其后，而 `masterpiece, best quality` 等画质标签放置于 prompt 尾部的格式。
  - **Standard Anime:** 通用的 Stable Diffusion 二次元质量提示词模板。
- **画面分级与搜索过滤 (Rating & Search Scope):**
  - 提供 Safe (全年龄)、Sensitive (微存)、NSFW (限制级)、Explicit (露骨) 四档评级，自动转换对应模型的评级词。
  - 支持“全局搜索”与“当前分类搜索”的范围切换，对 28 个原生分类标签库执行高效的实时过滤。
- **交互式提示词构建 (Prompt Builder):**
  - **选择标签:** 点击左侧标签胶囊，即可将其添加到 Builder 流程中。如果该标签带有预设权重，系统会自动解析。
  - **调整权重:** 支持对已添加标签进行可视化增减权重操作 (0.1 ~ 2.5 步长 0.1)。
  - **移动排序:** 支持点击“向前/向后”移动调整标签在整个提示词中的顺序。
  - **实时预览与一键复制:** 实时生成并展现 Positive (正向) 与 Negative (负向) 提示词，支持一键独立复制并提供 Toast 视觉反馈。
- **中文本地化翻译与中英文双向搜索 (Chinese Translation & Bilingual Search):**
  - **混合翻译引擎:** 扫描构建时结合了 static 翻译字典（`byzod/a1111-sd-webui-tagcomplete-CN`）和基于常见词汇表（`vocab`）下划线切分的词组翻译算法。新增对年龄后缀（如 `12_years_old` -> `12岁`）与 20 种特定分类后缀（如 `_theme` 主题、`_animal` 动物、`_creature` 生物、`_environment` 环境、`_lighting` 光照、`_tattoo` 纹身、`_style` 风格等）的规则匹配，智能解析前缀并拼接对应分类中文，同时扩充了视角、面料、国籍、年龄等维度的专有基词字典，显著提高专业画风与特定领域标签的本地化翻译覆盖度。
  - **双语混合检索:** 检索栏同时匹配英文标签原词与其中文翻译，支持输入中文、英文或首字母模糊搜索。
  - **可视化中文对照:** 标签胶囊及已选列表中直观显示中文对照译名（形如 `long_hair (长发)`），方便直观对比与权重调整。
  - **分类中英文双语显示:** 提示词分类的侧边栏、搜索框提示、以及已选标签胶囊对应的分类，均以中英文双语对照格式显示（形如 `饰品/配饰 (Accessory)`）。
- **智能灵感与多风格随机推荐 (Smart Inspiration & Style Recommendation):**
  - **风格分类:** 支持 12 种预设生成风格的随机推荐 (🎲 随机任意风格、🌸 日系动漫、📸 写实人像、🌃 赛博朋克、🔮 奇幻魔法、🎋 国风华服、🚀 科幻太空、🍃 清新田园、👥 多人合照、⛓️ 束缚调教、🎭 隐奸推荐、🔞 羞羞推荐)。
  - **多风格过滤规则:** 针对每种风格定义专属的强关联标签作为底座（如赛博朋克的 neon lights, cyberpunk 等），并引入排除黑名单（如多人合照排除 `1boy`，日系二次元排除写实词等）和特异的 Nudity (暴露度) 级联展开逻辑，全自动为 Prompt Builder 点选并组装高质量的风格词，一键赋能激发 AI 绘图灵感。

### 3.7 AI 提示词大师模块 (AI Prompt Master)
为 Z-Image, Flux 等对自然语言理解非常好的前沿生图模型，提供基于大模型扩展的创意提示词扩充服务：
- **多预设服务商配置:**
  - **Ollama (本地运行):** 预设本地 API 服务地址（`http://localhost:11434/v1`），支持自定义本地运行的模型（如 `llama3`, `qwen2.5` 等），零网络开销、纯本地离线运行。
  - **DeepSeek (云端调用):** 提供超高性价比的远程模型调用配置预设。
  - **OpenAI / 自定义:** 支持任何标准兼容 OpenAI SDK 的第三方中转站及自定义大语言模型 API 接口。
  - **连线测试:** 提供一键测试大模型服务联通性状态，并以直观的色彩微动效（Success / Failed / Error Message）对连线进行状态诊断反馈。
- **配置信息本地持久化:**
  - 所有的 API 地址、密钥及模型配置完全存储于用户浏览器本地的 `localStorage`，无需后端服务器，保证个人及企业生成创意的完全私密性。
- **互动性参数化设置:**
  - **Step 1 - 选择艺术风格:** 预置 10 类大热生图风格（赛博朋克、日系动漫、写实人像、奇幻魔法、电影质感、水彩手绘、古典油画、3D C4D渲染、国风水墨、蒸汽朋克），智能拼接对应风格的底层指令集。
  - **Step 2 & 3 - 构图与光影设定:** 快速组合构图视角（如半身、俯视、电影特写）与光影氛围（如体积光、黄金时刻、霓虹冷色），形成完整的生图指导架构。
- **AI 智能生成与中英双语对照:**
  - **长文本自然语言优化:** 大模型将用户的简单创意脑洞，智能转化组合为高画质、描述细致的段落式英文 Prompt。
  - **中英双语对照输出:** 独立展示英文 Prompt 与大模型对该段 Prompt 的中文意译解释，免除用户反复调用第三方翻译的心智损耗。
  - **一键独立复制:** 提供正向提示词和中文释义的分离一键复制功能。
  - **友好的本地调试帮助:** 调用出错时（如 Ollama 未开启或模型未装载），系统能够智能检测并给出包含 Ollama 本地命令行指令在内的详实排查排错指引。

### 3.8 视觉视频生成模块 (Video Generator)
为兼容阿里云 HappyHorse、通义万相 (Wanx) 等前沿视频生成大模型，系统内置了功能完备的视频生成管理界面：
- **主流视频模型全兼容、按模式精准隔离与自定义模型扩展 (Multi-Model, Mode Filtering & Custom Expansion):**
  - 阿里 HappyHorse 1.1 专有视觉模型 (HappyHorse-1.1-T2V / HappyHorse-1.1-I2V / HappyHorse-1.0-R2V / HappyHorse-1.0-Video-Edit)。
  - 通义万相 2.1 高清/极速与 2.0 序列模型 (wanx2.1-t2v-turbo, wanx2.1-t2v-plus, wanx2.1-i2v-turbo, wanx2.0-t2v)。
  - 智谱 CogVideoX 5B (cogvideox-5b) 及 腾讯 Hunyuan 视频模型 (hunyuan-video)。
  - **原生大模型精简收录 (10款指定模型):** 系统原生内置并严格收录用户指定的 10 款最新大模型（T2V: `qwen-image-2.0-pro-2026-06-22`、`qwen-image-2.0-pro-2026-04-22`、`wan2.7-t2v-2026-06-12`、`wan2.7-t2v-2026-04-25`、`happyhorse-1.1-t2v`；I2V: `wan2.7-i2v-2026-04-25`、`happyhorse-1.1-i2v`；R2V: `wan2.7-r2v-2026-06-12`、`happyhorse-1.0-r2v`；Edit: `happyhorse-1.0-video-edit`），并在模型下拉框按模式隔离过滤。
  - **自定义模型拓展与删除管理:** 支持用户通过“+ 管理模型”按需创建任意新模型 ID 与标签，绑定当前选择的生成模式并使用 `localStorage` 持久化，自动同步至主选框与画廊筛选菜单；同时支持在模型选择器旁及管理弹窗内一键删除个人自定义模型，系统原生内置模型保持受保护状态。
  - **宽屏容器对齐与操作区拓展 (Layout Alignment & Workspace Expansion):** 统一页面容器与顶部导航栏的最大宽度规格至 `1600px`，实现全站顶级顶栏与内容面板两侧精准齐平；将视频生成控制面板宽度拓展至 `420px`，为模型管理与参数配置提供充裕的操作排版空间。
  - **顶部导航按钮字体属性与图标全量对齐 (Navbar Typography & Icon Consistency):** 规范顶部“选择本地目录 / 载入默认库”操作按钮（`.nav-action-btn`）的 CSS 规则，使其 `font-family`（`Outfit`）、`font-size`（`0.95rem`）及 `font-weight`（`500`）与导航 Tab 选项 100% 保持一致；同时为所有顶部导航页签（首页、分类媒体库、提示词生成器、AI 提示词大师、视频生成）统一配备视觉对应的 Lucide 小图标，保持美观和谐的基调。
- **全模式生成支持与精确媒体数量控制 (Generation Modes & Input Boundaries):**
  - **文生视频 (T2V):** 纯文本 Prompt 驱动场景运动，无需且不展示图像上传框 (0 张图像)。
  - **图生视频 (I2V):** 支持上传 1 张源图 (`image1`) 进行动态预测。
  - **参考生视频 (R2V):** 支持 1~9 张多图参考矩阵 (`image1` ~ `image9`) 上传与管理；预设多图对白模版 (如 `The princess image2 was imprisoned in bedroom and be threaten by the dragon man image1. They had a long dialogue.`)。
  - **视频编辑 (Edit):** 支持同时上传 1 个待编辑原视频 (`video1`) 与 1 张编辑参考图 (`image1`)；预设视频重绘与对象替换模版 (如 `参考image1，将video1中正在行驶的白色邮轮替换为图中所示的太空飞船...`)。
- **高自由度动效参数调优与快捷变量标签 (Flexible Controls & Variable Insertion):**
  - **按模式按需变量标签插入:** 提示词框下方根据模式精准提供 `+ video1`、`+ image1` ... `+ image9` 变量快捷插入按键。
  - **按模式智能切换 Prompt 预设:** 根据选中的生成模式（t2v / i2v / r2v / edit）提供专属的示例 Prompt 预设与 🔄 随机换一批功能。
  - **分辨率与画面比例:** 支持 720P / 1080P 清晰度，以及 16:9、9:16、1:1、4:3、3:4 等 5 种主流画面长宽比。
  - **视频编辑时长与声音模式配置 (Edit Duration Alignment & Audio Settings):** 视频编辑模式下提供“与输入对齐”选项按键（自动与原视频 `video1` 播放长度 100% 同步）与 3~15 秒自定义 slider 调节；提供**声音设置**（智能生成 `auto` / 保持原声 `origin`）两项控制按键，配合带悬浮 Popover 的帮助提示图标。
- **云端 DashScope API 接入、跨域代理与异步任务轮询 (API, CORS Proxy & Async Task Polling):**
  - **符合官方规范的 Media 数组与 Ratio 参数 Payload 请求:** 严格遵照阿里 DashScope 官方协议解析媒体项（R2V/Edit 的参考图组装为 `{ type: "reference_image", url: ... }`，原视频组装为 `{ type: "video", url: ... }`，I2V 源图组装为 `{ type: "first_frame", url: ... }`）；画面长宽比统一以 `parameters.ratio`（如 `16:9`）下发；编辑模式下对齐时长时省略 `duration` 字段；R2V 预设与变量药丸全面适配 `[Image 1]`、`[Image 2]` 方括号官方语法。
  - **CORS 跨域代理:** 在 Vite 配置中搭建 `/api/v1` 与 `/dashscope-proxy` 本地代理层（指向阿里 MAAS / DashScope 域名），彻底消除浏览器直连 API 产生的预检 CORS 阻塞问题。
  - **异步任务轮询机制:** 在调用 `POST` 合成视频服务获取 `task_id` 后，系统自动发起周期为 10 秒的平缓异步轮询请求（对 `GET /api/v1/tasks/{task_id}` 执行代理获取），实时同步任务状态（`PENDING` -> `RUNNING` -> `SUCCEEDED` / `FAILED`），避免请求过于频繁。
  - **成果视频自动落盘存入当前工作目录 `/vedio`:** 视频渲染成功后（`status === "SUCCEEDED"`），系统自动通过 File System Access API 句柄或本地文件导出机制，将 `.mp4` 文件自动写入当前活动工作目录的 `/vedio/`（或 `/media/vedio/`）子目录下，并同步更新媒体画廊（`images`）分类库。
