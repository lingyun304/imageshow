# PromptMedia - 媒体分类展示网站需求文档 (PRD)

## 1. 项目背景与目标
随着 ComfyUI 和 Stable Diffusion 等 AI 创作工具的普及，设计师与多媒体创作者累积了海量的媒体文件（包含图片、音频和视频）。这些图片的 PNG/WebP 元数据中嵌入了极具价值的“生成参数”（工作流、提示词等），且音视频多媒体的管理也需要更优雅的呈现方式。
本项目旨在构建一个**完全去后端的、视觉效果极其 premium 的静态媒体展示系统**，让用户通过简单的本地文件夹分类组织媒体文件，并自动解析和展示图片中的 ComfyUI 生成细节，以及播放视频和音频，方便内容管理、技术重现 and 成果分享。

---

## 2. 系统技术架构
为了实现无后端的全静态运行并支持本地文件同步，系统采用以下架构：
- **静态资源扫描器 (Build-time CLI):** 使用 Node.js 编写的 `scan-media.js` 脚本。运行时扫描本地 `/public/media/`，解析 PNG (`tEXt`/`iTXt`) 或 WebP (`XMP`) 二进制数据块中的 ComfyUI 提示词数据，并提取侧边栏 `.json` 文件作为兼容格式。生成最终的 `/public/media-data.json` 索引文件。
- **现代化前端展现层 (React Single Page App):** 采用 React 19 + Vite 8 + Vanilla CSS 构建，纯静态展示，通过 `fetch()` 读取扫描生成的 JSON 数据库。
- **UI/UX 设计语言:** 支持三套独立的视觉系统一键无缝切换：默认的【科技暗黑 (Cyber Dark)】、亮色温暖且具有春意感的【清新雅致 (Fresh Mint)】、以及搭载了动感渐变背景与丰富微交互的【炫酷动感 (Dynamic Anime)】。三套主题均融入了高品质玻璃态拟物化 (Glassmorphic) 模块、响应式瀑布流 (Masonry Grid) 和顺滑的过渡动画，提供极佳的感官体验。

## 3. 功能需求规格

### 3.1 首页介绍模块 (Home)
- **品牌 Hero 区:** 采用高辨识度的标题、大字号渐变文本和科技微动效，确立高级、现代的整体基调。
- **实时统计看板:** 动态显示已解析的本地图片总数、分类统计数、以及 ComfyUI 图像元数据解析覆盖率。
- **核心卖点卡片:** 突出“零后端托管”、“自动文件夹分类”、“ComfyUI 流程提取”等主要特性。
- **分类导航预览:** 自动渲染每个分类的卡片，背景显示该分类下的首张图片，点击可瞬间跳转到对应画廊。

### 3.2 分类媒体库模块 (Gallery)
- **多维度检索与搜索:**
  - 支持对正向提示词、负向提示词、模型名、采样器和文件名的模糊搜索。
  - 支持按生成所用大模型（Checkpoint）、采样器（Sampler）等参数的多选组合筛选。
  - 支持仅过滤显示“有 ComfyUI 生成数据”的图片。
- **动态分类 Tab 栏:** 根据文件夹层级自动生成分类选项卡，并带有一键清除所有筛选功能。
- **瀑布流展示 (Masonry Layout):** 图片以等宽、不等高的响应式瀑布流卡片平铺，悬浮时露出快速参数卡与“查看生成参数”按钮。

### 3.3 参数详情查看器 (Detail Modal)
- **超限遮罩与按键导航:** 模糊背景弹出层，支持使用键盘左右方向键 (←/→) 切换上一张/下一张图片，支持 `Esc` 退出。
- **极清预览与下载:** 左侧展示大图，支持自适应缩放，提供原图一键下载。
- **正/负向提示词视窗:** 卡片式展示提示词，包含一键复制按钮与绿色成功状态提示。
- **参数矩阵宫格:** 规范整理展示 Steps、CFG、Seed、Sampler、Scheduler、Denoise、Checkpoint 以及分辨率与文件属性。
- **源码级工作流查看器 (Tabs):** 
  - **Prompt API Graph:** 提取并美化展示底层的 API 格式节点网络。
  - **Workflow UI Graph:** 提取展示 ComfyUI 界面布局连线数据。
  - 均支持一键复制，用户可直接贴回 ComfyUI 恢复生成场景。

### 3.4 部署说明页 (Guide)
- 集成在网站内部的部署使用文档，包含本地扫描流程、文件夹结构图、元数据提取原理解释和各平台静态部署指引，自带锚点侧边导航。

### 3.5 多主题视觉系统 (Theme System)
为了契合不同用户的审美并适配不同的生成图片风格，系统设计并实现了三套独立的视觉风格，支持在导航栏右侧一键切换：
1. **科技暗黑 (Cyber Dark - 默认):**
   - **设计基调:** 赛博朋克与科幻感。
   - **色彩系统:** 极黑背景 (`#07090e`)、深蓝紫色卡片、紫色 (`#8b5cf6`) 和青色 (`#06b6d4`) 霓虹流光。
   - **核心视觉:** 高反差暗度、玻璃磨砂、明亮的边框发光，体现出高级且冷静的工业范。
2. **清新雅致 (Fresh Mint):**
   - **设计基调:** 纯净、自然、明亮与简约。
   - **色彩系统:** 温暖的浅绿乳白色背景 (`#f5faf7`)、纯白卡片背景、森林薄荷绿 (`#10b981`) 和天蓝色 (`#0ea5e9`) 辅助色。
   - **核心视觉:** 柔和边缘投影、淡绿细微边框、春意盎然的护眼色调，适合日式或大自然类生成图展示。
3. **炫酷动感 (Dynamic Anime):**
   - **设计基调:** 动漫次元、幻彩与强烈的感官动效。
   - **色彩系统:** 深紫色基底、流光溢彩的霓虹艳粉 (`#ec4899`) 与赛博金黄 (`#eab308`)。
   - **核心视觉:** 
     - **动态渐变背景:** 四色流光渐变（`linear-gradient`）缓慢平滑循环滚动，具有呼吸感。
     - **浮动粒子动效:** 首页 Hero 区域渲染多组随机缓动上浮的毛玻璃幻影气泡。
     - **卡片倾斜动画:** 卡片悬浮时触发 3D 旋转偏转（`rotate(0.6deg)`）与弹性微放大，极具动感。
     - **按钮抖动与闪烁:** 核心操作按钮及分类 Tab 享有炫目的呼吸阴影和微抖动（`shake`）反馈。

### 3.6 提示词生成器模块 (Prompt Generator)
为了方便创作者构建高质量的提示词，系统集成了一个强大的 Danbooru 提示词生成器：
- **画质前缀预设 (Model Presets):**
  - **Pony Diffusion V6:** 自动带入通用的 `score_9, score_8_up, score_7_up...` 高画质评分前缀与 `source_anime` 标签，以及契合的分级标签。
  - **Illustrious XL:** 支持角色与版权（Character/Copyright）标签强制置前，分级标签紧跟其后，而 `masterpiece, best quality` 等画质标签放置于 prompt 尾部的格式。
  - **Standard Anime:** 通用的 Stable Diffusion 二次元质量提示词模板。
- **画面分级与搜索过滤 (Rating & Search Scope):**
  - 提供 Safe (全年龄)、Sensitive (微存)、NSFW (限制级)、Explicit (露骨) 四档评级，自动转换对应模型的评级词。
  - 支持“全局搜索”与“当前分类搜索”的范围切换，对 28 个原生分类标签库执行高效的实时过滤。
- **交互式提示词构建 (Prompt Builder):**
  - **选择标签:** 点击左侧标签胶囊，即可将其添加到 Builder 流程中。如果该标签带有预设权重，系统会自动解析。
  - **调整权重:** 支持对已添加标签进行可视化增减权重操作 (0.1 ~ 2.5 步长 0.1)。
  - **移动排序:** 支持点击“向前/向后”移动调整标签在整个提示词中的顺序。
  - **实时预览与一键复制:** 实时生成并展现 Positive (正向) 与 Negative (负向) 提示词，支持一键独立复制并提供 Toast 视觉反馈。
