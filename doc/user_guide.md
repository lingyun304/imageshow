# PromptMedia - 平台使用与操作指南

本手册将引导您如何在本地添加媒体、运行扫描更新、以及在网站前台操作搜索、播放和提取 ComfyUI 参数。

---

## 1. 本地媒体分类与管理
本画廊通过您的**本地文件夹结构**来实现分类管理。

### 1.1 目录规范
打开您的项目文件夹，定位到 `public/media/` 目录（若是打包后的 `dist/` 部署包，则对应 `dist/media/` 目录）：
1. **新建分类:** 在 `media/` 下新建任意名称的子文件夹，该文件夹的名字将直接成为网站上的“分类标签”。
2. **放置媒体:** 将 ComfyUI 生成的媒体（图片或音视频）直接拖入对应的子文件夹中。
   - 例如：
     - `public/media/cyberpunk/` -> 存放科幻、霓虹风格的媒体。
     - `public/media/portraits/` -> 存放人物肖像媒体。
     - `public/media/nature/` -> 存放自然风光媒体。

### 1.2 兼容侧边栏模式 (JPG/编辑过的图)
如果您的媒体被压缩过（例如导出的 JPG，或使用 Photoshop 保存而丢失了元数据），本画廊提供**同名 JSON 侧边栏方案**：
- 在媒体同级目录下，放置一个与媒体同名的 `.json` 文件。
  - 例如，媒体为 `nature/mountain.jpg`，则在该目录下新建 `nature/mountain.json`。
- JSON 内容可以是 ComfyUI 导出的 API 格式，如：
  ```json
  {
    "prompt": {
      "3": {
        "inputs": { "seed": 123456, "steps": 20, "cfg": 7.0, "sampler_name": "euler" },
        "class_type": "KSampler"
      }
    }
  }
  ```
- 扫描器在扫描时会自动融合该 JSON 的数据。

## 2. 数据库同步与目录切换 (双击可执行程序)
每当您在本地媒体文件夹中**添加、删除或重命名**媒体后，或者**想要切换展示另一个媒体文件夹**时，可以通过运行扫描程序来同步前台数据库。

我们已经将扫描程序编译打包为了双击即用的本地应用（存放于项目根目录的 `bin/` 文件夹下）：
- 🖥️ **Windows 用户**: 直接双击运行 `bin/scanner-win.exe`。
- 🍎 **macOS 用户**: 在终端中运行 `bin/scanner-macos`（首次运行如遇权限拦截，可在系统的“安全性与隐私”中允许运行，或在终端执行 `chmod +x bin/scanner-macos` 赋予执行权限）。

### 2.1 首次运行：绑定与切换图片目录
1. **启动程序**：双击运行 `bin/scanner-win.exe`。
2. **输入路径**：程序会检测到是首次运行，并在控制台中提示：
   ```text
   请输入文件夹路径: 
   ```
   此时，您可以**直接将想要展示的 ComfyUI 图像输出文件夹拖入到该窗口中**（或者手动复制输入文件夹的绝对路径，例如 `D:\ComfyUI\output`），然后按下**回车键**。
3. **软链接绑定**：扫描程序会自动在项目目录下的 `public/` 内创建一个名为 `images` 的软链接（Junction），直接指向您的目标文件夹。**这个过程不需要占用任何额外的磁盘空间**。
4. **保存配置**：路径会被自动记录在项目根目录下的 `directory-config.json` 配置文件中。

### 2.2 日常运行：自动加载与一键更新
- 绑定完成后，以后每次需要更新画廊，您只需要**直接双击运行** `bin/scanner-win.exe`（或运行 `bin/scanner-macos`），程序会自动加载上一次的目录，并在数秒内完成图片和元数据解析，解析完成后会提示“按下回车键即可退出”。同时，也会自动运行 `build-tags.js` 更新提示词标签数据库。

### 2.3 如何更换绑定的图片目录？
如果您希望把画廊切换到另一个文件夹，有以下两种方式：
- **方式 A (推荐)**：打开命令行窗口，加上参数运行程序：
   ```bash
   # 使用 Node 运行
   node scan-media.js --switch
   
   # 或者运行二进制程序时加参数
   bin/scanner-win.exe --switch
   ```
- **方式 B (最简单)**：直接删除项目根目录下的 `directory-config.json` 配置文件，下一次启动扫描程序时，就会重新提示您输入/拖入新的路径。

## 3. 网页前端直连切换目录 (无需运行 EXE / 免安装 Node)
除了通过双击运行 `scanner-win.exe` 扫描同步媒体外，您还可以直接在**网页前端载入并切换您电脑上的任意本地文件夹**：

1. **选择本地文件夹**：在画廊网页顶部导航栏，点击 **“选择本地目录”** 按钮。
2. **授权访问**：在浏览器弹出的文件夹选择器中，选中您想查看的 ComfyUI 输出媒体目录（或包含分类子文件夹的媒体文件夹），在浏览器弹出权限提示时点击 **“允许访问”** (只读授权)。
3. **前端实时解析 (极速)**：
   - 网页应用会直接在浏览器端读取文件流，解压 PNG 的压缩数据块并解析 ComfyUI 原始媒体及伴随 `.json` 的参数。
   - 解析完成后，画廊会自动刷新类别并呈现媒体，**全程在浏览器沙盒中本地完成，绝对安全且不耗费网络流量**。
4. **恢复默认扫描库**：若您想切换回原先由可执行程序扫描出的默认数据库，只需点击导航栏的 **“载入默认库”** 按钮即可一键复原。

> [!NOTE]
> 由于浏览器的沙盒安全限制，在网页前端直连的文件夹授权在**页面刷新后会失效**（再次浏览需重新选择）。如需永久在线部署展示，请使用 `scanner-win.exe` 扫描保存生成。

## 4. 前端交互操作说明

### 4.1 画廊浏览与快捷切换
- **首页统计:** 首页显示实时的媒体统计看板，点击下方的分类卡片可以直接进入对应的分类媒体库。
- **分类 Tab:** 进入画廊后，顶部 Tab 栏可以快速切换查看特定分类，并实时显示每个分类下的媒体数量。

### 4.2 高级筛选与全局搜索
- **全局搜索框:** 在顶部输入关键字，可瞬间模糊匹配正向提示词、负向提示词、模型名、采样器名称和文件名，检索速度极快。
- **高级筛选抽屉:** 点击搜索框右侧的“筛选”按钮，会展开筛选面板：
  - **按大模型筛选:** 选择下拉框中解析出的模型文件（如 `sd_xl_base_1.0.safetensors`），仅看该模型生成的图。
  - **按采样器筛选:** 选择特定的采样算法（如 `euler_ancestral`）。
  - **仅元数据过滤:** 勾选“仅显示含有 ComfyUI 元数据”可自动屏蔽普通无参数的媒体。
  - 点击“重置筛选”一键恢复默认状态。

### 4.3 详情弹窗与参数提取
点击任意一个媒体的卡片，会唤醒毛玻璃详情模态窗：
- **快捷键支持 (极其高效):** 
  - 使用键盘 **左方向键 (←)** 切换上一个媒体。
  - 使用键盘 **右方向键 (→)** 切换下一个媒体。
  - 按下 **ESC 键** 退出弹窗。
- **提示词复制:** 点击“复制提示词”或“复制”，系统会自动将对应的正向/负向提示词载入系统剪贴板，按钮背景会变为绿色并显示“已复制”。
- **参数卡片:** 直观地展示种子 (Seed)、采样器、步数、CFG、大模型名字、分辨率和文件大小。
- **源码工作流复制 (高级用户):**
  - 在详情窗右侧最下方，有 `Prompt API Graph` (API 节点) 和 `Workflow UI Graph` (UI 工作流) 两个选项卡。
  - 用户可在此查看最原始的 JSON 结构，并点击右上角复制按钮直接复制 JSON。
  - 复制的 JSON 文本可以在 ComfyUI 界面中通过 **Load** 按钮加载，或者直接拖入/粘贴至 ComfyUI 画布，即可 100% 还原该图 the 连线和参数，方便复现或共享！

### 4.4 主题切换与视觉风格
网站在导航栏右侧提供了一个**三色胶囊主题切换器**。点击相应图标即可瞬间变换全站的视觉风格。系统会自动将您的选择记住在本地（LocalStorage），下次打开网站时将自动应用上次选择的主题：
1. **🌙 暗黑:** 科技深邃的赛博朋克风。以深蓝色和紫色霓虹流光为主体，配有极具质感的毛玻璃磨砂面板与发光阴影。
2. **🍃 清新:** 舒适明亮的护眼薄荷风。采用温暖的奶白/淡绿背景与森林绿点缀，阴影更为柔和细腻，整体排版干净轻快。
3. **⚡️ 动感:** 极富次元特色的活力动感风。采用四色流光渐变背景及毛玻璃幻彩气泡，卡片具备 3D 偏转和形变。

### 4.5 提示词生成器助手
点击顶部导航的 **“提示词生成器”** 即可启动基于本地分类标签库的 Prompt 助手：
1. **选择大模型风格**:
   - **Pony Diffusion V6**: 自动添加通用的 score 评分高画质前缀和 rating、source 等辅助元标签，自动注入推荐的负向提示词起手式。
   - **Illustrious XL**: 强制执行二次元最优架构，让 Character 与 Copyright 类标签排在最前面，评级标签次之，而 masterpiece 与 best quality 质量词则追加置于 prompt 末尾。
   - **Standard Anime**: 提供通用的 Stable Diffusion 提示词架构。
2. **画面分级设定**: 自由切换 Safe (全年龄)、Sensitive (微存)、NSFW (限制级)、Explicit (露骨)，程序将自动替换输出为所选模型对应的专用分级词。
3. **标签分类点选、中文翻译与双语搜索**:
   - **分类栏**: 左侧快速切换 28 个原生分类，了解每个分类下的可用标签总数，分类名称已全部进行中文友好汉化（例如“hair”汉化为“头发/发型”）。
   - **中英双语检索**: 搜索框完美支持中英文双向搜索。您可以键入英文标签原词，亦可键入中文（如“长发”、“红脸”），系统进行毫秒级双语过滤匹配。
   - **双语对照展示**: 标签胶囊以 `英文原词 (中文对照)` 的形式展示（如 `long_hair (长发)`），悬浮时还可查看完整的分类和翻译浮窗。
   - **智能复合翻译**: 支持对含有特定后缀（如 `_theme`、`_animal`、`_environment`、`_lighting` 等 20 类修饰词）和年龄模式（如 `12_years_old`）的标签进行自动拆分、语义转换与拼装翻译，保证更专业、顺畅的双语对照。
   - **已选 Builder 对照**: 选入右侧的标签卡片依然保留中文翻译，方便您在调节权重、前后移位排序时进行直观的心智比对。
   - **标签胶囊**: 单击左侧的标签胶囊即可瞬时填入右侧 Builder 中，再次点击可一键退选。
4. **提示词精细化修饰 (Builder)**:
   - **权重微调**: 在右侧 Builder 的标签卡上，直接点击 `+` 或 `-` 按钮，可以 0.1 为步长在 0.1 ~ 2.5 范围内精密调节单个词的权重（自动输出为 WebUI / ComfyUI 标准权重格式）。
   - **位置移动**: 点击 `◀` 或 `▶` 按钮，可将选中的提示词前移或后移，严格控制词的位置对生成结果的影响。
   - **一键独立复制**: 正向和负向提示词分屏实时拼接预览，双复制按钮均具备防抖及状态通知反馈。
