# PromptGallery - 平台使用与操作指南

本手册将引导您如何在本地添加图片、运行扫描更新、以及在网站前台操作搜索和提取 ComfyUI 参数。

---

## 1. 本地图片分类与管理
本画廊通过您的**本地文件夹结构**来实现分类管理。

### 1.1 目录规范
打开您的项目文件夹，定位到 `public/images/` 目录：
1. **新建分类:** 在 `images/` 下新建任意名称的子文件夹，该文件夹的名字将直接成为网站上的“分类标签”。
2. **放置图片:** 将 ComfyUI 生成的原图（PNG 或 WebP）直接拖入对应的子文件夹中。
   - 例如：
     - `public/images/cyberpunk/` -> 存放科幻、霓虹风格的图片。
     - `public/images/portraits/` -> 存放人物肖像图片。
     - `public/images/nature/` -> 存放自然风光图片。

### 1.2 兼容侧边栏模式 (JPG/编辑过的图)
如果您的图片被压缩过（例如导出的 JPG，或使用 Photoshop 保存而丢失了元数据），本画廊提供**同名 JSON 侧边栏方案**：
- 在图片同级目录下，放置一个与图片同名的 `.json` 文件。
  - 例如，图片为 `nature/mountain.jpg`，则在该目录下新建 `nature/mountain.json`。
- JSON 内容可以是 ComfyUI 导出的 API 格式，如：
  ```json
  {
    "prompt": {
      "3": {
        "inputs": { "seed": 123456, "steps": 20, "cfg": 7.0, "sampler_name": "euler" },
        "class_type": "KSampler"
      }
    }
  }
  ```
- 扫描器在扫描时会自动融合该 JSON 的数据。

## 2. 数据库同步 (图片扫描)
每当您在本地 `images/` 文件夹中**添加、删除或重命名**图片后，都需要运行扫描脚本以更新前台数据库。

1. 打开终端（PowerShell 或 CMD），进入项目根目录。
2. 运行扫描命令：
   ```bash
   npm run scan
   ```
3. 控制台会输出扫描日志，例如：
   ```text
   🔍 Scanning local folders under public/images...
   ✅ Scanned: [cyberpunk] cyberpunk_street.jpg (Metadata: Yes)
   ✅ Scanned: [nature] forest.png (Metadata: Yes)
   🎉 Success! Scanned 2 images and wrote data to public/images-data.json
   ```
4. 扫描完成后，刷新网站页面即可看到最新的图片和参数。

## 3. 前端交互操作说明

### 3.1 画廊浏览与快捷切换
- **首页统计:** 首页显示实时的图片统计看板，点击下方的分类卡片可以直接进入对应的分类画廊。
- **分类 Tab:** 进入画廊后，顶部 Tab 栏可以快速切换查看特定分类（如 "Nature" 或 "Cyberpunk"），并实时显示每个分类下的图片数量。

### 3.2 高级筛选与全局搜索
- **全局搜索框:** 在顶部输入关键字，可瞬间模糊匹配正向提示词、负向提示词、模型名、采样器名称和文件名，检索速度极快。
- **高级筛选抽屉:** 点击搜索框右侧的“筛选”按钮，会展开筛选面板：
  - **按大模型筛选:** 选择下拉框中解析出的模型文件（如 `sd_xl_base_1.0.safetensors`），仅看该模型生成的图。
  - **按采样器筛选:** 选择特定的采样算法（如 `euler_ancestral`）。
  - **仅元数据过滤:** 勾选“仅显示含有 ComfyUI 元数据”可自动屏蔽普通无参数的图片。
  - 点击“重置筛选”一键恢复默认状态。

### 3.3 详情弹窗与参数提取
点击任意一张图片的卡片，会唤醒毛玻璃详情模态窗：
- **快捷键支持 (极其高效):** 
  - 使用键盘 **左方向键 (←)** 切换上一张图片。
  - 使用键盘 **右方向键 (→)** 切换下一张图片。
  - 按下 **ESC 键** 退出弹窗。
- **提示词复制:** 点击“复制提示词”或“复制”，系统会自动将对应的正向/负向提示词载入系统剪贴板，按钮背景会变为绿色并显示“已复制”。
- **参数卡片:** 直观地展示种子 (Seed)、采样器、步数、CFG、大模型名字、分辨率和文件大小。
- **源码工作流复制 (高级用户):**
  - 在详情窗右侧最下方，有 `Prompt API Graph` (API 节点) 和 `Workflow UI Graph` (UI 工作流) 两个选项卡。
  - 用户可在此查看最原始的 JSON 结构，并点击右上角复制按钮直接复制 JSON。
  - 复制的 JSON 文本可以在 ComfyUI 界面中通过 **Load** 按钮加载，或者直接拖入/粘贴至 ComfyUI 画布，即可 100% 还原该图的连线和参数，方便复现或共享！

### 3.4 主题切换与视觉风格
网站在导航栏右侧提供了一个**三色胶囊主题切换器**。点击相应图标即可瞬间变换全站的视觉风格。系统会自动将您的选择记住在本地（LocalStorage），下次打开网站时将自动应用上次选择的主题：
1. **🌙 暗黑:** 科技深邃的赛博朋克风。以深蓝色和紫色霓虹流光为主体，配有极具质感的毛玻璃磨砂面板与发光阴影。
2. **🍃 清新:** 舒适明亮的护眼薄荷风。采用温暖的奶白/淡绿背景与森林绿点缀，阴影更为柔和细腻，整体排版干净轻快。
3. **⚡️ 动感:** 次元梦幻的活力动感风。采用四色平滑缓动的渐变流光背景，配有随机上浮的渐变背景气泡。卡片悬浮时具备 3D 偏转和弹性形变交互，核心操作按钮随渐变发光呼吸，并伴随微动作抖动反馈，动感十足。
